# Identicons

identicons service

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## browser access http://127.0.0.1:3000/identicon/{param}

For example: http://127.0.0.1:3000/identicon/helloworld.bit

## pm2 deployment
```bash
npm run start:pm2
```
